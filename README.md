# Poisonous Mushroom Prediction

Goal: Predict whether a mushroom is **poisonous or edible** using the provided dataset.

## Data
Place the following files in `./data`:
- `train.csv`
- `test.csv`
- `sample_submission.csv`

## Environment
1. Create venv: `python3 -m venv .venv && source .venv/bin/activate`
2. Install deps: `pip install -r requirements.txt` (or use the list in this README)

## Dependencies
- pandas
- numpy
- scikit-learn
- matplotlib
- jupyter
